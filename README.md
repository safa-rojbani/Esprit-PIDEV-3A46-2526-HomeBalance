# HomeBalance – Family Task Management Platform

## Overview
HomeBalance is a web platform designed to help families manage household responsibilities through task assignment, validation, and reward systems.

This project was developed as part of the PI Project – 3rd Year Engineering Program at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

## Features
- Task assignment to children
- Task validation with photo proof
- Reward and score system
- Parent dashboard
- Family task management

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Twig

### Backend
- Symfony 6
- PHP
- MySQL
- Doctrine ORM

## Architecture
The application follows a modular architecture with separate modules for tasks, scoring, notifications, and family management.
